Intellectual Sharing

The goal of this website is to organize all of the world's ideas in a meaningful and comprehensive way, using a graph database and some simple rules. It is a 100% community supported open source knowledge base where anyone can contribute and learn.

  • Want to add some meta nodes (aka TypeNode s) ? You can do so with the TypeNode Editor
  • Want to view some nodes? You can do so with links like this: /viewNode/actor/Daniel Craig/
  • Check out the database the login is neo4j/django, and you can see the database with this query: match (n) return n
